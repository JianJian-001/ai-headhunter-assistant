# 岗位材料输入约定

## 建议字段

| 字段 | 必填 | 说明 |
|---|---|---|
| `company_name` | 是 | 客户公司全称 |
| `job_title` | 是 | 招聘岗位名称 |
| `jd_text` | 是 | 原始 JD 或岗位说明 |
| `city` | 否 | 工作地点 |
| `salary_range` | 否 | 薪酬预算 |
| `business_background` | 否 | 业务背景、团队背景 |
| `team_notes` | 否 | 顾问补充说明 |

## 输出提取重点

1. 岗位名称和业务线
2. 核心职责
3. 必须项
4. 可协商项
5. 材料中的模糊点
6. 建议继续追问的问题
