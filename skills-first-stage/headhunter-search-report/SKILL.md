# 做单秘籍

## 作用边界

这个技能负责岗位深度理解与寻访策略，不负责招聘站抓取、不负责猎聘搜人、不负责单份简历主评分。

适用：

- 需要把 JD 和补充材料转成寻访策略
- 需要明确岗位商业背景、组织要求、人才画像
- 需要输出接单建议、目标公司方向、建议追问

不适用：

- 直接找候选人
- 输出公司情报整包
- 输出候选人推荐报告

## 输入

- `company_name`
- `job_title`
- `jd_text`
- 可选：客户补充说明、行业背景、组织信息、预算信息

## 工作流

1. 读取 [report-framework.md](./references/report-framework.md)
2. 结合 [做单秘籍.md](../../../做单秘籍.md) 的业务口径整理输入
3. 按五维分析框架输出分节结论
4. 在结尾补 [follow-up-prompts.md](./references/follow-up-prompts.md) 中对应的建议追问
5. 需要本地文件时，按 [report-template.md](./assets/report-template.md) 生成 Markdown 正文

## 输出要求

默认交付：

1. 岗位概览
2. 五维分析框架
3. 目标公司方向
4. 接单建议
5. 建议下一步追问

## 严格禁止

1. 不调用飞书、Dify、RAGFlow 或其他远端系统
2. 不编造目标公司、薪酬行情和客户背景
3. 材料不足时必须明确依据边界
4. 不输出无结构的大段堆砌文本
