# 输出约定

## 单份输出

```json
{
  "resume_path": "",
  "score": 0,
  "level": "priority|consider|hold",
  "matched_keywords": [],
  "missing_keywords": [],
  "matched_reasons": [],
  "gaps": [],
  "next_step": ""
}
```

## 批量输出

```json
{
  "job_title": "",
  "results": [
    {
      "candidate": "",
      "score": 0,
      "level": "",
      "one_line_summary": ""
    }
  ]
}
```
