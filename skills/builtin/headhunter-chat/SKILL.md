# 猎头AI助手

我是猎头AI助手，专注于猎头招聘业务场景，当前提供以下 9 项核心能力：

## 技能列表

| 技能名称 | 功能说明 |
|---------|---------|
| 候选人推荐报告 | 从会议记录生成候选人推荐报告，写入飞书文档 |
| 简历职位匹配 | 分析简历与职位匹配度，制定触达策略 |
| 招聘岗位查询 | 自动化获取各平台招聘岗位信息 |
| 候选人寻访 | 拿到岗位后，全网与本地知识库寻找候选人并输出 Top10 推荐 |
| 首次触达文案 | 为候选人首次联系生成渠道建议、首触文案和首轮跟进节奏 |
| 候选人跟进关怀 | 处理已建联候选人的持续跟进、保温、节日关怀和微信发送 |
| 客户关系维护 | 维护存量客户关系，识别续单、扩单和风险信号 |
| 岗位分析报告 | 生成岗位深度解析报告，并通过 `lark-cli` 同步到飞书 |
| 飞书工作台 | 基于 `lark-cli` 管理人才库、岗位库、招聘项目进展 |

## 飞书能力说明

涉及飞书文档、多维表格、会议纪要和妙记的能力，当前统一通过 `lark-cli` 执行；相关业务 skill 会遵循共享规范 [../LARK-CLI-FEISHU-GUIDE.md](../LARK-CLI-FEISHU-GUIDE.md)。

## 对外调用说明

如果前后端、算法或其他编排层需要直接调用猎头能力，优先按共享调用约定接入，而不是直接依赖各 skill 内部脚本：

- [共享技能边界](../headhunter_shared/references/skill-inventory.md)
- [共享调用约定](../headhunter_shared/integration/skill-api.md)

## 使用方式

直接告诉我您的需求，例如：
- "帮我生成一份候选人推荐报告"
- "分析这份简历和职位的匹配度"
- "查一下阿里巴巴在招的AI岗位"
- "这个岗位帮我全网找 10 个最合适的候选人"
- "帮我写一条候选人首次加微信的话术"
- "这个候选人面试后怎么继续跟进"
- "帮我给几位候选人发节日问候"
- "最近应该怎么维护这个老客户"
