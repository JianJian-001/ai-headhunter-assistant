# 数据格式参考

## 预览文件

- **文件名**：`greeting/YYYY-MM-DD-{001-999}.md`
- **存储位置**：技能根目录下的 `greeting/`
- **生命周期**：生成预览 → 用户微调 → 读取发送 → 追加明细

### 预览内容格式

```markdown
### 发送前预览

| # | 候选人 | 发送内容 |
|---|--------|---------|
| 1 | 张大千 | 大千，节日快乐~ ... |
| 2 | 简致 | 简致，节日快乐~ ... |
```

### 明细格式

```markdown
### 发送数据明细

总 **N** 人 | 成功 **N** 人 | 失败 **N** 人 | 成功率 **X%**

| # | 候选人 | 发送内容 | 状态 | 时间 |
|---|--------|---------|------|------|
| 1 | 张大千 | 大千，节日快乐~ ... | 已发送 | 2026-03-12 10:36:54 |
| 2 | 简致 | 简致，节日快乐~ ... | 未找到 | - |
| 3 | 简致 | 简致，节日快乐~ ... | 已发送 | 2026-03-12 10:38:10 |
```

### 对话输出

```text
已完成，明细数据已更新: [greeting/YYYY-MM-DD-001.md](file://<实际绝对路径>)
```

## send 命令格式

```bash
python3 scripts/main.py send "简致" "简致，植树节快乐~ ..." "植树节"
python3 scripts/main.py send "简致" "简致，最近还好吗~ ..." "日常关怀"
```

参数顺序：`send "名称" "内容" "主题"`。
