# 候选人跟进与关怀打法

## 适用阶段

- 首次联系已建立
- 候选人正在了解机会
- 候选人进入推荐前后
- 面试前后
- Offer 前后

## 跟进目标

- 保持候选人在线
- 降低流失风险
- 推进到下一节点
- 记录更完整的候选人信息

## 常见动作

### 轻跟进

适用于：
- 刚加上微信
- 对方表示可以了解
- 需要确认下一个沟通时间

### 价值型跟进

适用于：
- 候选人回复变慢
- 需要提升岗位吸引力
- 需要补充团队、业务、薪酬、发展空间等价值信息

### 节点关怀

适用于：
- 面试前提醒
- 面试后安抚
- Offer 前后保温
- 节日或职业变化节点的自然触达

## 关怀与推进平衡

- 过于频繁会造成打扰
- 只关怀不推进会失去节奏
- 只推进不关怀会让候选人感到被销售

理想状态是每次触达同时兼顾：
- 关系温度
- 推进目的
- 信息沉淀
